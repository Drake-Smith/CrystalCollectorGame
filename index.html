Instructions

Each crystal represents a random number between 1-12. When you click on a crystal, the Current Total will update and add whatever that crystal's point value is.

You win when you click on the crystals until the Current Total matches the Number Goal. If you go over, you lose.

Remember, with each new game, the point value of each crystal changes to another random number between 1-12.
